## 🎯 Problem

Civil engineers spend hours manually searching through hundreds of pages 
of Egyptian (ECP) and Saudi (SBC) engineering standards to find specific 
code requirements. No existing tool understands engineering context 
or supports Arabic queries.

## 💡 Solution

CivilMind is a bilingual RAG-powered AI assistant that lets engineers 
ask questions in Arabic or English and get accurate, sourced answers 
from engineering codes instantly.
